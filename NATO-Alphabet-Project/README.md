# NATO Alphabet Project

A simple Python program that converts a word into its NATO phonetic alphabet equivalent.

## Example

Input:

```text
Hello
```

Output:

```text
['Hotel', 'Echo', 'Lima', 'Lima', 'Oscar']
```

## Requirements

- Python 3.x
- pandas

## Installation

```bash
pip install pandas
```

## Run

```bash
python main.py
```
