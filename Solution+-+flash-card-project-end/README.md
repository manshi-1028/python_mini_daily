# Flash Card App

A French-to-English flash card application built with Python and Tkinter.

## Features

- Random French flash cards
- Automatically flips after 3 seconds
- Tracks learned words
- Saves progress in words_to_learn.csv

## Technologies

- Python
- Tkinter
- Pandas

## Project Structure


## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
