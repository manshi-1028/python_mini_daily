# 🗣️ Kanye Quotes Generator

A simple Python GUI application built with Tkinter that fetches random Kanye West quotes using the Kanye REST API.

## Features

- Fetches a random Kanye quote with one click.
- Uses the Kanye REST API.
- Built with Python and Tkinter.
- Displays quotes on a custom background.

## Installation

Clone the repository.

```bash
git clone https://github.com/manshi-1028/python_mini_daily.git
```

Move into the project folder.

```bash
cd python_mini_daily/kanye-quotes-end
```

Install the dependency.

```bash
pip install -r requirements.txt
```

Run the project.

```bash
python main.py
```
